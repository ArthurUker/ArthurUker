# Renkang Guo — Translational Science & Data-Driven R&D Portfolio

Hi, I'm **Renkang Guo**. I build digital systems for **laboratory informatics, scientific data workflows, and cross-functional R&D coordination**.

My focus is turning fragmented experimental and operational processes into structured, reproducible, and decision-support-oriented products.

## What I work on

- Laboratory information workflows and sample/data traceability
- Scientific and operational dashboards for decision support
- Reproducible data handling and workflow automation
- Cross-team tools that connect research, operations, and reporting

## Selected projects

- **Tianjiabing_foodtestlab** — web-based food safety lab workflow and reporting system
- **project-management** — R&D project management platform for planning and coordination
- **sample-data-hub** — collaborative sample data sharing and editing system
- **digifluidic-brochure-builder** — scientific communication and technical brochure generation
- **food_safety_test_project** — food safety project planning and outreach presentation

## Technical strengths

- Frontend: JavaScript, TypeScript, HTML, CSS
- Data and systems: SQL, structured workflows, traceable records
- Delivery: documentation, iteration, maintainability, practical deployment

## Why this profile

I use software to make R&D and lab workflows easier to understand, easier to share, and easier to act on.
