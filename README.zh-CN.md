# 郭仁康 / Renkang Guo, Ph.D.

## 转化分子诊断｜数字微流控｜数据驱动研发系统

我是一名具有生物医学工程博士背景的研发人员，主要工作方向位于 **分子诊断、数字微流控、核酸检测平台和数据驱动研发流程** 的交叉领域。

我的工作重点是将实验室研发中的复杂实验流程、样本信息、检测结果、项目协同和技术文档转化为更加结构化、可追溯、可复现和便于决策的工作流。我同时具备核酸提取、扩增检测、数字微流控平台开发等 wet-lab 研发经验，以及实验室信息化、样本数据管理、研发项目协同和技术文档系统建设经验。

我尤其关注 **生物学、工程技术与数据系统之间的交叉界面**，希望通过结构化数据流程和数字化工具，提高转化研发过程中的效率、透明度、复现性和产品化执行能力。

---

## 关于我

我的专业背景覆盖分子诊断、样本前处理、数字微流控和实验室信息化系统。

我具备以下方面的经验：

- 核酸提取流程开发，包括磁珠法核酸提取相关流程
- qPCR 检测体系开发与优化
- LAMP 和 CRISPR 相关核酸检测流程
- 数字微流控 sample-to-answer 平台开发
- 实验问题排查与分析性能评估
- 实验室样本数据、检测结果和报告流程的结构化管理
- 试剂、仪器、软件和技术文档之间的跨职能研发协同

在实验平台开发之外，我也构建轻量级数字系统，用于支持研发项目管理、样本数据管理、技术文档整理、检测报告生成和研发流程追踪。这些系统的目标是让实验室和产品化研发流程更加清晰、可复现、可协同和可执行。

**基于网页的实验室流程与报告系统**

- 有助于提高实验室流程的可追溯性、报告效率和流程透明度
- 项目意义：展示了如何将实验室实际运行需求转化为网页化的数据管理和报告系统

**关键词：** 实验室流程、样本追踪、结构化报告、食品安全检测
