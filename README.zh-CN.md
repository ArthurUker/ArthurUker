# 郭仁康｜转化科学与数据驱动研发作品集

你好，我是 **Renkang Guo**。我专注于构建面向 **实验室信息化、科研数据流程、研发协同管理** 的数字化系统。

我的目标是把分散的实验与运营流程，整理成结构化、可复用、可追踪、能辅助决策的产品。

## 我主要做什么

- 实验室信息流程与样本/数据追踪
- 支持决策的数据看板与业务可视化
- 可复用的数据处理与工作流自动化
- 连接研发、运营和报告的跨团队工具

## 代表项目

- **Tianjiabing_foodtestlab** —— 食品安全实验室业务流程与报告系统
- **project-management** —— 研发项目管理平台，用于计划和协作
- **sample-data-hub** —— 协作式样本数据共享与编辑系统
- **digifluidic-brochure-builder** —— 科研传播与技术宣传资料生成工具
- **food_safety_test_project** —— 食品安全项目规划与展示内容

## 技术能力

- 前端：JavaScript、TypeScript、HTML、CSS
- 数据与系统：SQL、结构化流程、可追溯记录
- 交付：文档、迭代、可维护性、实际部署

我用软件帮助研发和实验室流程更容易理解、更容易共享，也更容易真正落地。
