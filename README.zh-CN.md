## 转化方向


我致力于在转化研究与产品开发的 **生物–数据–技术界面** 上发展职业，特别是：


- 诊断、精准医学或个性化医疗公司的 **转化科学角色**
- 生物技术或医疗器械公司中管理数据流程与研发协调的 **研发信息化领导**
- 学术转化中心或政府研究机构的 **研究科学家职位**
- 结合分子诊断、生物标志物发现与数据驱动平台开发的 **应用研究角色**


我的长期目标是在 **分子诊断、定量测量系统、可重复数据流程和转化决策** 等方面发展深厚专业能力，同时成长为能在生物、工程、数据科学和面向产品团队之间有效沟通的多语言科学家。
